SENSOR ULTRASSONICO DE NÍVEL DE ÁGUA

--Descrição:
O sensor ultrassonico de nível de água utiliza ondas sonoras para detectar o quão próximo a água ou algum objeto está. Ele emite um som e mede a resposta do mesmo, assim podendo detectar com precisão o quão próximo a água e/ou algum objeto estão.

--REQUISITOS DE HARDWARE:
1x Placa arduino UNO
1x Sensor de distância ultrassônico
3x LED
3x Resistores 220 kOhm

--REQUISITOS DE SOFTWARE:
Arduino IDE

--INSTALAÇÂO:
Para utilizar este projeto, você precisará de:

-Um ambiente de desenvolvimento de sistemas de arduinos e o codigo fornecido dentro do link para o tinkercad abaixo neste documento.
-Os materias listados em "Requisitos".

1.Monte os sistemas apresentados nas imagens e no link da simulação.
2.Conecte o arduino em um computador que possui o arduino IDE e faça o upload do código mostrado no link de simulação.
3.Mantenha o sensor longe de qualquer objeto e vá aproximando e afastando para medir sua distância.
